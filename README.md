# WPF Resource files for all Syncfusion WPF assemblies 

The repository now includes a **resource file** specifically designed to support **localization** of static text elements used within the controls of the assemblies listed below. This file enables developers to easily adapt the user interface text to different languages and regions, enhancing internationalization support across the application.

As part of the initial setup, the resource file contains the **default locale**, set to "en-US" (English - United States).
